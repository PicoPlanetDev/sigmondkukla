Grades Dashboard Google Action

This Action allows you to ask Google Assistant about your grades. It retrieves grade information from Powerschool SIS and displays it as part of a conversation with Google Assistant.

I used Glitch to host this project for development and testing. I recently finished adding login functionality using Google Sign-In account linking. A separate webpage, available at https://dashboard-api-web.glitch.me makes POST requests to separate routes on my endpoint that control user data, such as creating, editing, and deleting users.

Every time a request is made, the user's Google account email is sent as a token to the endpoint. It is then decrypted with Google's public key and checked for authenticity. Then, the user's PowerSchool data is retrieved from the database, their grades are parsed from that, and the endpoint returns a response.

On my first attempt to host my webhook endpoint, I used PythonAnywhere. However, the endpoint must make requests to a Powerschool URL that is not on the PythonAnywhere whitelist because there are many different schools with their own URLs. With Glitch, this is not a limitation, however, the server must be kept alive using a repeated GET request to keep responses snappy.

Through building and deploying this, I learned:

  • Using Google Actions Console
  • Creating a Flask application
  • Parsing JSON
  • Hosting a Python application
  • Using Glitch
  • Using Postman to test requests
  • Creating a SQLite database
  • Creating a web settings interface
